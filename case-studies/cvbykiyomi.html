Project Overview

CV by Kiyomi is a resume builder designed with one thing in mind: helping job seekers in Nigeria and across Africa create standout CVs quickly and without paying for expensive tools. The problem with most resume builders is that they were designed for western markets, with formats, fonts and layout conventions that do not always read well in local hiring contexts.

This project was built entirely using vibe coding. I used Claude to generate the interactive form logic, template rendering and download functionality. The design was guided by my understanding of what a clean, readable CV actually looks like, and the AI handled translating that into functional code.

The Problem

Most job seekers do not have access to clean, professional CV templates. They either use Microsoft Word templates that look dated, pay subscription fees for tools like Canva Pro, or rely on a friend who knows design. None of these options feel built for them.

CV by Kiyomi was designed to close that gap. A free, simple, beautiful tool that anyone with a phone or laptop can use to build something they are proud to send to a recruiter.

Template System

Classic Template

Clean, single column layout. Works universally across industries. Strong typographic hierarchy with clear section breaks.

Modern Template

Two column layout with an accent sidebar. Better for creatives and tech roles. More visual personality while staying professional.

User Flow

USER JOURNEY
User lands on homepage
  → Sees clean headline and template previews
    → Clicks "Start Building"
      → Fills in personal information
        → Adds work experience (add/remove as needed)
          → Adds education
            → Adds skills
              → Picks a template style
                → Previews CV in real time
                  → Downloads as PDF
                    → Done

Design Decisions

The form is the product, so it needed to feel frictionless. Each section is clearly labeled, inputs have obvious purpose, and the user can see their CV taking shape in a live preview on the right side of the screen. This immediate feedback loop makes the process feel rewarding rather than tedious.

The output templates were designed with print in mind as much as screen. Typography is set at sizes that work on a physical page. Margins are generous. Color is used sparingly so the CV prints cleanly in black and white without losing its structure.

Outcome

CV by Kiyomi is live and accessible for free. It represents a meaningful step toward building genuinely useful tools for the Nigerian job market using AI assisted development. The same concept that would have required a development team and weeks of work was built and deployed as a real working product by one person using vibe coding.
