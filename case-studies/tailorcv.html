Project Overview

Tailor CV is a tool that solves one of the most frustrating parts of job hunting: the mismatch between what a job description asks for and what a generic CV says. Every recruiter will tell you that tailored applications perform dramatically better than generic ones. But most job seekers do not have the time or confidence to rewrite their CV for every single application.

Tailor CV bridges that gap. Paste in the job description, paste in your current CV, and the tool analyzes the match and suggests specific improvements. It is the kind of tool that turns a frustrating, time consuming process into something that takes five minutes.

The Problem

Applicant tracking systems reject a huge portion of CVs before a human ever reads them, simply because the language in the CV does not match the keywords in the job description. This is a known problem in the hiring industry but most job seekers have no practical way to address it without spending significant time manually reworking their documents for every application.

The result is that talented people get filtered out not because they are underqualified but because their CV was not written in the right language for that specific role. Tailor CV was designed to fix that at scale, for free, using AI.

How It Works

01
Paste the Job Description

Copy the full job description from wherever you found the role. Paste it into the left input field on the tool.

02
Paste Your Current CV

Paste in the text content of your existing CV. The tool reads the language you are currently using and compares it to what the role is asking for.

03
Analyze the Match

The tool highlights keywords present in the job description that are missing from your CV, and identifies sections that could be strengthened for this specific role.

04
Apply the Suggestions

Use the output to update your CV. The tool gives you the language and framing. You make the edits. Then apply with confidence.

Design Decisions

The interface was kept deliberately minimal because the user is already dealing with two documents. Adding visual complexity on top of that would create friction at exactly the wrong moment. The layout is two input panels with a clear action to analyze and a clean output section below.

Color coding in the output makes it easy to see at a glance which keywords are present, which are missing and which sections need the most work. The warm amber palette reinforces the sense of action and energy without feeling alarming or clinical.

Vibe Coding Process

This was one of the more technically interesting vibe coding projects because the core functionality required text comparison logic. I described the matching algorithm I wanted to Claude in plain language and worked through several iterations to get the output format right. The AI generated the JavaScript logic, I tested the edge cases and we refined together until the tool behaved consistently.

The entire experience of building Tailor CV using AI tools to build a tool that helps users work with AI output felt appropriately circular in a way I enjoyed. The project was live within a day of starting.

Outcome
